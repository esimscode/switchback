# Recommend a resume version

The user maintains five resume versions. Map the role to the version whose focus matches the role's center of gravity:

- **Master** — only when no targeted version fits, or the user asks for the full picture.
- **Platform-DevSecOps** — platform engineering, DevSecOps, infrastructure automation, SRE-adjacent roles.
- **Cloud-Infrastructure** — cloud engineer, cloud support, systems/infrastructure engineering, data center-adjacent roles.
- **Software-AI Automation** — software engineering, internal tools, AI/automation engineering, workflow automation roles.
- **Cybersecurity** — security analyst, SOC, cybersecurity intern, IT security support roles.

Always explain the choice in one or two sentences: why this version's framing serves this role, what to emphasize on it, and what not to overclaim for this particular posting.
