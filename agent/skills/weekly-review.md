# Weekly career review

1. Gather the week's picture: `retrieve_career_memories`, `list_projects`, and the career profile. Consider active applications and any follow-ups due.
2. Summarize honestly in four short sections:
   - **Moved** — what actually progressed (applications sent, milestones shipped, assets created).
   - **Stalled** — what hasn't moved and why it might be stuck.
   - **Due** — follow-ups, decisions waiting (saved applications sitting undecided), and project milestones.
   - **Next** — the 1-3 highest-leverage actions for the coming week, each concrete and finishable.
3. Keep it under 300 words. Practical beats comprehensive.
4. Save the review with `create_reflection` (type: weekly_checkin) so the log builds a weekly record.
